# TouchDX-DiagnosticTool

TouchDX 项目的 PC 端网络与输入诊断工具。

此工具主要用于帮助玩家测试：
- 安卓客户端与 PC 之间的网络连通性。
- 触控数据的接收和解析状态。

## 如何使用

1. 打开本工具。
2. 确保手机和电脑在同一局域网下，或已配置好 ADB 端口转发（推荐 `adb reverse tcp:4321 tcp:4321`）。
3. 工具将显示本机的 IP 地址，手机端输入该 IP（若是 ADB 转发则输入 `127.0.0.1`）。
4. 在手机端进行触摸，工具界面会实时反馈当前的连接状态和接收到的操作数据。

## 如何构建

1. 使用 Visual Studio 2022 或是 .NET 命令行工具打开本项目。
2. 还原 NuGet 依赖（如有）。
3. 编译运行即可。
